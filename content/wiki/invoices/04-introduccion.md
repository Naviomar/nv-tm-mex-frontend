# Introducción
El módulo **CFDI – Solicitudes y Pagos** permite gestionar facturas CFDI de proveedores y solicitudes de pago relacionadas.  
Integra funciones de control para cumplir con regulaciones fiscales y facilitar la conciliación contable.

# Componentes principales
- **Supplier CFDI Module Links** → Acceso rápido a los módulos de CFDI de proveedores.
- **Factura CFDI Request Payment** → Solicitudes de pago vinculadas a facturas CFDI.

# Funcionalidades
- Creación y consulta de solicitudes de pago.
- Integración con facturación electrónica CFDI.
- Control administrativo de facturas de proveedores.
