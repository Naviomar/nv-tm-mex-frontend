# Introducción
El módulo **Admin Pack** centraliza la administración de CFDI vinculados con facturas TM.  
Incluye exportación de facturas pendientes, carga de archivos comprimidos (ZIP) con CFDI y consulta de históricos.

# Submódulos disponibles
- **Export TM Invoices** → Exporta facturas pendientes de TM para ligarlas con CFDI.
- **Upload Zip File with CFDIs** → Permite cargar archivos ZIP que contengan CFDIs emitidos.
- **Zip Files Uploaded History** → Consulta el historial de cargas de archivos ZIP y su procesamiento.

# Funcionalidades
- Automatización en la exportación de facturas pendientes.
- Integración masiva de CFDIs mediante carga de archivos comprimidos.
- Consulta del historial para auditoría y trazabilidad.
