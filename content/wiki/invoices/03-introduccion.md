# Introducción
El módulo **Freight Forwarder Agent Invoices** permite consultar y capturar facturas emitidas por agentes freight forwarders.  
Se integra al sistema de cuentas por pagar, asegurando control sobre proveedores internacionales.

# Submódulos disponibles
- **F.F. Agent Invoices** → Consulta y captura de facturas emitidas por agentes freight forwarders.

# Funcionalidades
- Registro de facturas de intermediarios logísticos.
- Visualización centralizada para auditoría y control financiero.
