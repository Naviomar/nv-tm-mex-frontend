# Bienvenido al Wiki de Manuales

Este espacio contiene la documentación oficial de Naviomar TM México. Aquí encontrarás guías, manuales y mejores prácticas para el uso eficiente del sistema.

## ¿Qué puedes encontrar aquí?

- **Guías de inicio rápido:** Aprende a navegar y utilizar las funciones principales.
- **Manual por módulos:** Documentación detallada de cada módulo (Aéreo, Marítimo, Facturación, Catálogos, etc.).
- **Preguntas frecuentes (FAQ):** Respuestas a dudas comunes.
- **Solución de problemas:** Consejos para resolver incidencias frecuentes.
- **Glosario:** Definiciones de términos clave usados en el sistema.

## ¿Cómo usar este wiki?

Selecciona un módulo o tema en la barra lateral para acceder a la documentación correspondiente. Cada sección incluye capturas de pantalla, ejemplos y pasos detallados.

> Sugerencia: inicia por **Configuración → Airlines** para ver un ejemplo de documentación.
